Échelle d'espacement

Toutes les marges et paddings suivent une échelle en puissances de 2. En cas de besoin exceptionnel, rester sur un multiple de 2 (ex : 96px).

Règle : 8 → 16 → 32 → 64 → 128 → 256 → 512 · Multiples de 2 autorisés en secours

Anatomie d'une newsletter

La structure canonique, de haut en bas. Passez votre souris sur les blocs pour mettre en relation le schéma et les règles.

Header · Logo + Navbar
Hero image (cliquable)
Hero text + CTA
Modules (wildcards, texte/image)
Bloc réassurance
Paragraphe marque
Footer (2 colonnes)

Header

  • Logo + navbar, fond blue-primary ou blanc
  • Logo max-height : 128px
  • Hauteur respectant l'échelle d'espacement

Hero image

  • Toujours cliquable
  • Avec ou sans accroche textuelle
  • Flèche ou indice UI pour signaler la clickabilité

Hero text & CTA

  • Phrase principale + courte description
  • CTA link : le plus haut possible
  • CTA toujours au-dessus de la ligne de flottaison

Modules produits

  • Wildcards : nom seul, pas de prix
  • Chaque wildcard = lien direct vers la fiche produit
  • Préférer fonds clairs + packshots
  • Combinaisons texte/image autorisées avec CTA

Bloc réassurance

  • Comme sur le site (livraison offerte, etc.)
  • Icônes : max-height 96px
  • Typographie conforme au DS

Paragraphe marque

  • Logo + texte institutionnel (Depuis 1974…)
  • Placement avant le footer

Footer

  • Deux colonnes desktop, une sur mobile
  • Texte blanc sur fond blue-primary
  • Gauche : logo, adresse, réseaux
  • Droite : contact + boutique + copyright
